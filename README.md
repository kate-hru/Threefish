# Threefish
	
Проект состоит из 5 файлов: Threefish.java – непосредственная реализация самого симметричного блочного алгоритма шифрования, которая поддерживает три разных ключа: 256, 512 и 1024 бит; ThreefishCTR.java и ThreefishCFB.java – модули с реализациями режимов шифрования CTR и CFB соответственно; Main.java – консольный интерфейс для работы с остальными модулями проекта; ThreefishTest.java – программа, тестирующая все возможности шифратора с разными ключами, данными и режимами.
	
Примеры использования:	
		
		java Main -E secret.txt (Шифрование файла (ключ сгенерируется);
		
		java Main -E -p "пароль" file.txt.txt" (шифрование с паролем);
		
		java Main -E -r ./documents (шифрование директории);
		
		java Main -D file.txt.txt.enc (расшифрование (из .enc с метаданными));
		
		java Main -D -k 0011223344... -n AABBCCDD... file.txt.enc (расшифрование с ключом);
